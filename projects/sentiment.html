PySentiment

Technology stack: Flask, SQLAlchemy, PostgreSQL, HTML, Bootstrap CSS, Heroku Cloud

Try a live demo here

The machine learning application is written in Python and uses robust libraries including, but not limited to, pandas, sklearn, numpy and nltk.

Fundamentally, it is a logistic regression model trained on IMDb movie review dataset that attempts to correctly classify user-input reviews into positive and negative reviews.

The classifier is then serialized (using Pickle) and deployed to the website created using the Flask micro web-development framework. On the server, it's deserialized and then it's ready start predicting user sentiments.

The user then has the opportunity to tell the classifier whether or not the review has been correctly classified. This information (a (review_text, correct_label) tuple) is then stored in the backend built using PostgreSQL. Every fortnight, data from the database is used to update the classifier in the hopes of increasing its accuracy.
