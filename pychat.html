PyChat

Technology stack: Flask, SQLAlchemy, PostgreSQL, JavaScript, jQuery, HTML, Bootstrap CSS, Heroku Cloud

Try a live demo here

Description

PyChat is a full stack web application that supports user management, AI chatbot and a spreadsheet calculator.

To prevent (easier) DDOS attacks, PyChat requires users to provide (and confirm) their emails. It maintains a database of user information and allows for password change and reset. Users can also edit their profiles.

After signing up, users can chat with one another by creating chatrooms. Unfortunately, the realtime chat may have high latency as I'm not purchasing extra computing units from Heroku. The code can, however, be downloaded from GitHub and run on localhost.

If nobody is available, they can instead chat with an AI bot courtesy of Chatterbot.
Current work: Implementing RESTful API and learning how to host the application on a Linux box (probably some virtual private server).
